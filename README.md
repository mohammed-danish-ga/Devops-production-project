# Devops-production-project

This project demonstrates a complete containerized application deployment
using Docker, GitHub Actions CI/CD, and Kubernetes.

### Key Features
- Dockerized application
- CI/CD pipeline using GitHub Actions
- Kubernetes deployment with rolling updates
- Liveness and readiness probes
- Persistent storage using PV & PVC
- Dev and Prod environment separation using namespaces

### Tools Used
- Docker
- Kubernetes
- GitHub Actions
- Linux
